# csp-helper

Helpers for managing Content Security Policy (CSP) headers.

- Zero dependencies
- TypeScript support
- Preset CSP rules:
  - Datadog
  - Google Ads
  - Google Analytics 4
  - Google Tag Manager
  - Hotjar
  - ... and more

## Installation

```bash
npm install csp-helper
```
